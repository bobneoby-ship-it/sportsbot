HOW THE BOT WORKS
================================================================================

User asks: "Who won Portugal vs Morocco in 2026 World Cup quarterfinal?"

Bot process:
1. Searches Wikipedia for "2026 FIFA World Cup quarterfinal Portugal Morocco"
2. If not found → searches ESPN API
3. If not found → scrapes BBC Sport
4. If not found → uses DuckDuckGo search
5. Groq LLM analyzes the retrieved data
6. Returns: "Portugal won 3-0. Goals by: Silva (20), Ronaldo (45), Felix (78)"
7. Includes: "Source: Wikipedia + ESPN + BBC Sport"

KEY POINTS:
- No hardcoding of any match data
- No cached outdated information
- Always fetches live, current data
- Multiple sources ensure accuracy
- Falls back gracefully if primary source fails
- Groq provides intelligent analysis
